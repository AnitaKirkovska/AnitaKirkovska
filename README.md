### 🤖 What I'm building

The thread running through all of it: **one agent for one person, that actually remembers you.** Personal intelligence, not another centralized chatbot.

| Project | What it is |
|---|---|
| 🧠 [**vellumian**](https://github.com/AnitaKirkovska/vellumian) | A personal AI assistant that evolves with you. Memory, personality, proactive reach-outs across macOS, Telegram, and Slack. |
| 💻 [**dynamic-notch**](https://github.com/AnitaKirkovska/dynamic-notch) | Put your assistant inside your MacBook notch. Chat, hold-to-talk voice with streaming TTS, and a floating task bulb. |
| 🎲 [**kalshi-cli**](https://github.com/AnitaKirkovska/kalshi-cli) | A zero-dependency CLI for reading live odds and trading on Kalshi prediction markets. De-vigs any event into clean probabilities. |
| 🏟️ [**agent-yard**](https://github.com/AnitaKirkovska/agent-yard) | 30 days of AI agents. One small build a day, in public. |
| ✍️ [**seo-agent**](https://github.com/AnitaKirkovska/seo-agent) | An agent that analyzes keyword results, writes articles, and iteratively refines them against real criteria. |
